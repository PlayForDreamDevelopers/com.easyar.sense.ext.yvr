## 关于项目
这是 `com.easyar.sense.ext.yvr` 包的仓库。

EasyAR Sense Unity Plugin YVR 扩展。扩展 EasyAR 以配合 YVR Unity SDK 工作。

请阅读 [EasyAR 网站](https://www.easyar.cn) 上的文档了解如何使用。

### 示例
* Combination_BasedOn_YVR

## 要求
* com.easyar.sense (4000.0.0)
* com.yvr.core (1.25.1)
* com.yvr.enterprise (0.0.10)
