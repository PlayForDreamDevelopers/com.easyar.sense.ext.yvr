<div class="markdown-heading" dir="auto"><h2>About The Project</h2></div>

<p dir="auto">This is the repository for the <code>com.easyar.sense.ext.yvr</code> package.</p>
<p dir="auto">EasyAR Sense Unity Plugin YVR Extension. Extends EasyAR to co-work with YVR Unity SDK.</p>
